# .github
Community Health Files for the bdaim-bc-swat organization.

## Workflow TemplatesS
There are a number of workflow templates defined in this repository that can be used within the organization.  When adding new templates ensure the *.properties.json file's name includes one of the following categories and the appropriate SVG.

### Categories
- Build
- Deploy
- Test
- Run

### SVGs
The following SVGs should generally be used for each of the categories.  If a workflow uses a well known tool like terraform, a more specific SVG can be used but the template is free to use the standard SVGs as well.
#### Standard Category SVGs
- build
- deploy
- test
- run
